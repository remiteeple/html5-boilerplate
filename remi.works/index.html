Personal Website

I created and actively manage this website and the virtual private server it runs on. This website was built using HTML5, CSS3, JS & JQuery. I also integrated third party libraries like: Bootstrap, Isotope, CountUp and Tiny Slider. The backend of the website relies on a live Github repo that I push to when changes are ready. This website went through many, less tasteful iterations that helped better acquaint me with my own style. This site was designed with readability and usability in mind. Largely inspired by the Markdown Github projects and Gists I frequent, this site is an ode to simplicity and style. You may notice the distinct lack of colors on the site, this is intentional. My primary goal with the lack of color was to display my knowledge of conventional design philosophies without the need to rely on bright colors.

Again. This is the first website I've designed and coded from scratch so I do apologize if anything breaks or something isn't working quite right. If you notice an issue then please contact me.

C++ Game Engine

This nameless game engine was created for educational purposes. The engine itself is built on C++ and designed for Windows system architecture (though it could be very easily ported to Android). It uses GLSL to render meshes and has a complete game object management system alongside other core game and feedback (audio / visual) systems. The lighting is of custom design using self-made GLSL shaders. The physics system is driven by Bullet3D integration into the project. Sound uses XAudio.

This project is no longer being actively maintained by me, however I'd love to revisit it in the future to clean it up a bit and update some of the code to more modern C++ conventions.

AGE (Agnostic Game Engine)

AGE (Agnostic Game Engine) is a small project that I am currently working on individually. The premise is purely agnostic game platform that easily allows for swapping of core components, i.e. the renderer, entity management system, physics system, etc. The project was originally designed with DX12 in mind, however it currently remains free of any renderer until further development.

Stay tuned for any new updates to AGE!
